## Week 1 – History of Text and Writing
- Writing and numbers existed thousands of years before computers.
- Early writing systems like cuneiform were used mainly for accounting and record keeping.
- Writing systems can be used to represent multiple languages.

## Week 2 – Bits, Bytes, and Representing Data
- Computers store information using bits, which have two states: 0 and 1.
- A byte is made of 8 bits and is the basic unit of computer memory.
- Memory can be volatile (RAM) or non-volatile (SSD/HDD).

## Week 3 – Structure of Text
- Text is structured using lines, whitespace, and special characters.
- Markup languages like Markdown and HTML add structure and meaning to text.
- Special characters sometimes need to be escaped to be displayed literally.

## Week 4 – Manipulating Text and Text Editors
- Developers use text editors instead of word processors to work with code.
- Keyboard shortcuts and touch typing improve efficiency and accuracy.
- Editors like VS Code provide tools such as search, replace, and syntax highlighting.

## Week 5 – Command Line Interfaces
- A command line interface (CLI) allows users to interact with a computer using text commands.
- The command line is useful for speed, automation, and remote access.
- Common commands include ls, cd, pwd, cat, and less.
